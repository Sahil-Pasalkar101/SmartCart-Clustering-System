# SmartCart Customer Segmentation System

## Overview
Developed an intelligent customer segmentation system for SmartCart using
unsupervised machine learning techniques.

## Business Problem
SmartCart uses generic marketing campaigns for all customers, leading to:
- Inefficient marketing spend
- Poor customer targeting
- Difficulty identifying high-value customers
- Delayed churn detection

## Solution
Built a clustering-based customer segmentation system that groups customers
based on:
- Demographics
- Purchase behaviour
- Website engagement
- Loyalty indicators

## Dataset
- 2240 customer records
- 22 original attributes
- Features include demographics, spending, campaign responses and web activity

## Feature Engineering
- Age
- Customer Tenure
- Total Spending
- Total Children
- Education Categories
- Marital Status Categories

## Machine Learning
- K-Means Clustering
- Elbow Method
- Silhouette Score Evaluation

## Results
Identified distinct customer segments for personalized marketing and retention strategies.
